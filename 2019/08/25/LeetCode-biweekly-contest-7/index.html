1166. Design File System

考察基本的数据结构（树）和字符串处理。
对路径进行建树，每个文件或目录是一个节点，节点上有value。查找和增长的过程都是类似字典树。
时间复杂度: O(N).
空间复杂度: O(N).

1167. Minimum Cost to Connect Sticks

贪心算法。每次选择cost最小的2个stick进行connect操作。一个stick越早被connect，它的长度越可能影响整个cost。
