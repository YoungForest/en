1608. Special Array With X Elements Greater Than or Equal X

签到题。从小到大枚举可能的答案，进行检查。

时间复杂度: O(N * log N),

1609. Even Odd Tree

二叉树的层序遍历。使用广度优先搜索（BFS），记录层数和前一个元素即可。

时间复杂度: O(N),
空间复杂度: O(N).

1610. Maximum Number of Visible Points

滑动窗口。这里有2个技巧：

  1. 使用复数及其方法处理几何问题。可以快速计算向量的夹角。
  2. 把所有点 + 360度 遍历2次。以处理跨0问题。

还有一个需要注意的点是：当点和观察点重合时需要特殊处理。

时间复杂度: O(N log N),
空间复杂度: O(N).

1611. Minimum One Bit Operations to Make Integers Zero

参考群主的视频写的代码。比赛时没做出来，赛后看lee215的discuss也是讲的比较简单和跳跃。还是群主的视频好，把你当傻逼一样讲，速度还慢。虽然时长挺长的，但对于这种比较难的题目还是十分有效的。

时间复杂度: O(log N),
空间复杂度: O(log N).
